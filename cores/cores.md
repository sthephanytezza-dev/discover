# Cores

Usamoos CSS para alterar cores do nosso documento.

## Tipos

- background-color (para caixas)
- color (para textos)
- border-color (para caixas)
- outros...

## Valores

Podemos definir os valores por

- palavra-chave (blue, transparent)
- hexadecimal (#990011)
- funções: rgb, rgba, hsl, hsla

```css
element {
  /* Keyword values */
  color: currentColor;

  /* <named-color> values */
  color: red;
  color: orange;
  color: tan;
  color: rebeccapurple;

  /**/
}
```
